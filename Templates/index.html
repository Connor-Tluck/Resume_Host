Roadway Construction Survey Shot Geometry Fit

Personal Project

Current roadway design is done by shooting a digital survey which is visually interpreted by designers. Designers reference manuals to develop all aspects of roadway design per the scope of the project. By treating the survey shot data as the database that it is, designs may be able to be automated using established design criteria and various machine learning elements from built in python packages. Fitting program takes survey centerline shot data, classifies it as a curve shot or a tangent shot, then best fits curve and tangent geometry to the points, generating clean data tied to the best fit geometry. Further development will be to develop proposed roadway coorridors and evaluate for cost and safety. Following steps will be to iteratively generate designs and evaluate for optimal design.

CitiBike Ride Analysis and Mapping

Personal Project

NYC Citibike posts all user data to table formats with start/stop ride location data as well as trip duration, start and stop time. I downloaded my data for my first year in Manhattan. Journal Word Cloud Analysis

Personal Project

Personal Project in which I downloaded all journal entries over the last few years of my life. I began keeping a journal on and off when my father was diagnosed with cancer as a tool to self manage some of the emotional ups and downs at the time. I fed the text data into a regular expression library and created word clouds to look at trends of word frequency.
